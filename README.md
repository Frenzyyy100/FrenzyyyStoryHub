# NovéLore — Read Without Limits

A personal web-based novel reading platform built with love and carrot by **Frenzyyy**.

---

## About

NovéLore is a lightweight, mobile-friendly reading site for original stories. No ads, no accounts, no paywalls — just pure reading.

Currently hosting **The Lantern Pact: Prologue**, a Dark Fantasy series following three mismatched members of the Kaihon Light Pact as they navigate missions, secrets, and something far bigger moving in the shadows.

---

## Features

- Hero slider with auto-swipe
- Novel pages with chapter tracking
- Bookmarks that persist across visits
- Notifications system
- Light / Dark / System theme
- Reader preferences (font size, line spacing, width)
- Changelog page
- Fully mobile-friendly
